### 題目：進出管理系統
#### 內容：因武漢肺炎疫情關係做出一個管理系統，方便做進出管理
#### 功能性需求：紀錄進出、新增個人資料、查詢進出紀錄
    
#### 非功能性需求：使用者介面清楚明瞭、系統方便維護、查詢速度要快
